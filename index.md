# Introduction

RCL Cloud TnT is a hub for individuals and organizations embarking on a journey of digital transformation. Users will use modern cloud services such as Microsoft Azure and software development platforms such as ASP.NET to build projects.

## Live Events

There are regular live video events that users can attend online. These events provide hands-on demonstration on Cloud DevOps and Software Development. Events are broadly divided into two types:

### Hands-On Workshops

Demonstrations of practical activities. This is done on the main website.

### Discussion Sessions

Question and Answer sessions and general discussions. This is normally done with Microsoft Teams Meeting.

## Open Badges

RCL CloudTnT issues [Open Badges](./open-badges/open-badges.md) to users who demonstrate competence in Cloud DevOps and Software Development. These badges are digital credentials that are hosted online and shared by users on websites and social platforms.
